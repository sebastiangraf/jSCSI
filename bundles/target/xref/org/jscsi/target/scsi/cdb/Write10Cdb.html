27       * This variable represents the value of the DPO bit. Since the jSCSI Target
28       * does not support advanced caching strategies, the value of this variable
29       * is ignored.
30       * <p>
31       * A disable page out (DPO) bit set to zero specifies that the retention priority shall be determined by
32       * the RETENTION PRIORITY fields in the Caching mode page (see 6.4.5). A DPO bit set to one specifies that
33       * the device server shall assign the logical blocks accessed by this command the lowest retention
34       * priority for being fetched into or retained by the cache. A DPO bit set to one overrides any retention
35       * priority specified in the Caching mode page. All other aspects of the algorithm implementing the cache
36       * replacement strategy are not defined by this standard.
37       * <p>
38       * NOTE 11 - The DPO bit is used to control replacement of logical blocks in the cache when the
39       * application client has information on the future usage of the logical blocks. If the DPO bit is set to
40       * one, then the application client is specifying that the logical blocks accessed by the command are not
41       * likely to be accessed again in the near future and should not be put in the cache nor retained by the
42       * cache. If the DPO bit is set to zero, then the application client is specifying that the logical blocks
43       * accessed by this command are likely to be accessed again in the near future.
44       */

114     public Write10Cdb(ByteBuffer buffer) {
115         super(buffer);// OPERATION CODE + CONTROL
116 
117         // RDPROTECT
118         byte b = buffer.get(1);
119         writeProtect = (b >> 5) & 7;
120 
121         // DPO
122         disablePageOut = BitManip.getBit(b, 4);
123 
124         // FUA
125         forceUnitAccess = BitManip.getBit(b, 3);
126

128         forceUnitAccessNonVolatileCache = BitManip.getBit(b, 1);

130         // GROUP NUMBER
131         b = buffer.get(6);
132         groupNumber = b & 31;
133     }
